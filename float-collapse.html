Float Collapse

What I've Learned

Float is a property used to move block–level elements left or right. This causes the element to move to one side and the parent element's content wraps around it. It is removed from the flow in relation to other block elements. Removing an element from the flow removes the dimensions of it. This can cause float collapse.

Float collapse is when an element within another element is floating and the content breaks through the parent element, or container, dimension into other elements. When this happens, the content is visually out of place. This can be repaired by clearfix.
