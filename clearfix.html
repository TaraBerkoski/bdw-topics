Clearfix

What I've Learned

Clearfix is a method to avoid float collapse. Multiple techniques, codes, or hacks can avoid float collapse visually, but there are reasons to not use most of these.

Methods

Empty Div

This is the fastest and simplest way to fix a float collapse, but it is completely wrong semantically. The way to do this, even though you shouldn't, is to put in an empty div sectioning element after the element floated. It looks good, but it isn't. Don't do this.

Overflow

Another way to fix the problem is to set the parent element's overflow settings to auto. This works semantically, but is really meant to be used on elements of set sizes to create scrollbars instead of extremely lengthy element boxes.

Easy Clearing

The current most correct technique to resolve float collapse is to use an additional class onto the floated element. To do this, you must include the after psuedo selector to a class (in this case, clearfix). That class must then be added to the floating element.
