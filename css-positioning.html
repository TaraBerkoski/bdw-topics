CSS Positioning

What I've Learned

There are four positioning options in CSS. These include: static, relative, absolute, and fixed.

Static is the default position. This means that the elements flow as they normally would.

Relative moves the element in relation to the static position.

Absolute moves the element in relation to the closest ancestor element that has a relative (or absolute) position.

Fixed sets the element in relation to the viewport.
